MINHAS HABILIDADES.

JavaScript

Linguagem de programação usada para criar interatividade e dinamismo em páginas web, permitindo a manipulação de conteúdo e a resposta a ações do usuário em tempo real.

Css

Linguagem usada para estilizar e definir o layout de páginas web, controlando aspectos como cores, fontes, espaçamento e o posicionamento de elementos na tela.

Html

Linguagem de marcação fundamental para a criação de páginas web, estruturando o conteúdo com elementos como textos, imagens, links e formulários.

React

Biblioteca JavaScript desenvolvida pelo Facebook para construir interfaces de usuário interativas e eficientes, permitindo criar aplicações web dinâmicas por meio de componentes reutilizáveis.
